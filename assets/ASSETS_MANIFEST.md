## 📋 资源制作清单

### 设计原则
1. **视觉一致性** - 所有资源使用统一的配色方案和视觉风格
2. **品牌识别** - 突出五子棋核心元素
3. **清晰度** - 在小尺寸下仍可清晰辨认
4. **专业感** - 符合Steam平台质量标准

### 配色建议
- 主色调: 深蓝色或深灰色背景
- 强调色: 金色或白色棋子元素
- 辅助色: 棋盘木纹色调

### 文件命名规范
- 使用小写字母和下划线
- 避免空格和特殊字符
- 使用描述性名称

## 🛠️ 制作工具推荐

### 图标和截图
- **Figma** - 免费,在线协作
- **GIMP** - 免费开源,功能强大
- **Photoshop** - 专业级工具
- **Canva** - 模板丰富,易于上手

### 预告片
- **OBS Studio** - 免费录屏软件
- **DaVinci Resolve** - 免费专业视频编辑
- **Shotcut** - 免费开源视频编辑器
