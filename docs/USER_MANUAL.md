## 游戏简介

五子棋 · Gomoku 是一款现代化的五子棋游戏，提供精美的界面设计和智能的AI对手。无论你是新手还是高手，都能找到适合自己的游戏方式。

### 游戏特色

- 🎮 **三种游戏模式** - 人人对战、人机对战、练习模式
- 📋 **两种规则** - 经典规则和禁手规则
- 🤖 **三种AI难度** - 轻松、进阶、大师
- 🎨 **精美界面** - 现代化设计，流畅动画
- 💡 **智能提示** - AI辅助推荐落子
- 📊 **实时分析** - 局势判断和棋谱记录

## 快速开始

### 第一步：启动游戏

1. 打开命令行/终端
2. 进入游戏目录
3. 运行 `npm run serve`
4. 浏览器访问 `http://localhost:4173`

### 第二步：选择设置

在主界面选择：
- **游戏模式**：人人对战 / 人机对战 / 练习模式
- **规则**：经典规则 / 禁手规则
- **棋盘尺寸**：15×15 或 19×19
- **AI难度**（仅人机对战）：轻松 / 进阶 / 大师
- **执子颜色**（仅人机对战）：黑子 / 白子

### 第三步：开始游戏

点击"**开始游戏**"按钮，开始对局！

## 游戏规则

### 经典规则

最简单直接的五子棋规则：

1. **黑子先行**，双方轮流落子
2. **连五获胜**：横、竖、斜任意方向连成5子
3. **无禁手**：任何落子都是合法的
4. **平局**：棋盘下满无人连五则为平局

**适合人群**：新手入门，轻松对战

### 禁手规则（Renju）

更具竞技性的专业规则：

**黑方禁止的棋型：**

1. **三三禁手**
   - 同时形成两个或以上的"活三"
   - 活三：三子连线，两端都是空位

2. **四四禁手**
   - 同时形成两个或以上的"四"
   - 包括活四（四子两端空）和冲四（四子一端空）

3. **长连禁手**
   - 形成六子或更长的连线
   - 注意：只有黑方有禁手限制

**白方规则：**
- 无任何禁手限制
- 连五或六子以上均可获胜

**适合人群**：高手竞技，策略对战

### 胜负判定

- **连五即胜**：先连成5子的一方获胜（黑方禁手规则下）
- **对方认输**：点击"认输"按钮立即判负
- **超时判负**：无时间限制
- **平局**：棋盘满且无人连五

## 游戏模式

### 1. 人人对战

**适合场景**：两人面对面博弈

**特点**：
- 双方轮流在同一设备上落子
- 完整的悔棋和换边功能
- 实时棋谱记录

**操作流程**：
1. 选择"人人对战"模式
2. 选择规则和棋盘尺寸
3. 点击"开始游戏"
4. 黑方先落子，然后轮流
5. 先连五者获胜

### 2. 人机对战

**适合场景**：与AI对弈，练习棋艺

**特点**：
- 三种AI难度可选
- 可选择执黑或执白
- AI思考时间模拟真实对局

**AI难度**：

| 难度 | 特点 | 适合人群 | AI思考时间 |
|------|------|----------|-----------|
| 轻松 | 简单评估，随机选择 | 新手练习 | 0.3秒 |
| 进阶 | 中等评估，有策略 | 普通玩家 | 0.6秒 |
| 大师 | 深度搜索，高级策略 | 高手挑战 | 1.2秒 |

**操作流程**：
1. 选择"人机对战"模式
2. 选择AI难度和执子颜色
3. 点击"开始游戏"
4. 如果选择白子，AI先手
5. 轮到你时点击棋盘落子
6. AI会自动回应

### 3. 练习模式

**适合场景**：自由探索，无压力练习

**特点**：
- 无AI对手
- 可自由落子黑白双方
- 完整的悔棋功能
- 适合研究棋局

**操作流程**：
1. 选择"练习模式"
2. 选择规则和棋盘尺寸
3. 点击"开始游戏"
4. 自由在棋盘上落子
5. 可随时悔棋或重新开始

## AI难度说明

### 轻松难度

**算法特点**：
- 简单的棋型评估
- 从前6个候选位置随机选择
- 容易犯错

**适合人群**：
- 五子棋新手
- 想要轻松获胜
- 练习基本技巧

### 进阶难度

**算法特点**：
- 中等深度评估
- 考虑攻防策略
- 加权随机选择（前3个候选）
- 有一定威胁

**适合人群**：
- 有一定基础的玩家
- 想要提升棋艺
- 学习中级策略

### 大师难度

**算法特点**：
- 深度搜索和评估
- 高级棋型识别
- 精确计算
- 最优选择策略
- 很少犯错

**适合人群**：
- 五子棋高手
- 想要极限挑战
- 研究高级战术

**挑战建议**：
- 需要深思熟虑
- 注意防守
- 利用禁手规则（如果启用）

## 操作指南

### 基本操作

#### 落子
1. 等待轮到你落子
2. 将鼠标移到棋盘上
3. 会显示预览棋子（半透明）
4. 点击确认落子

#### 悔棋
- 点击"悔棋"按钮
- 人人对战：撤销最后一步
- 人机对战：撤销两步（你和AI各一步）

#### 提示
- 点击"提示"按钮
- AI会在棋盘上高亮推荐位置
- 显示黄色高亮提示

#### 换边
- 仅在开局前可用（未落子时）
- 切换你执黑或执白
- 人机对战模式专属

#### 重新开始
- 清空当前棋盘
- 保留所有设置
- 重新开局

#### 认输
- 立即结束对局
- 对方获胜
- 显示结算界面

#### 返回设置
- 回到主菜单
- 可以重新选择模式、规则等

### 快捷键

目前游戏不支持键盘快捷键，所有操作通过鼠标点击完成。

### 触摸屏操作

游戏完全支持触摸屏设备：
- 点击棋盘落子
- 点击按钮执行操作
- 响应式设计，自动适配屏幕大小

## 游戏技巧

### 基础技巧

#### 1. 开局布局
- **占据中心**：天元（棋盘中心）是最重要的位置
- **控制星位**：四个角星和边星是重要据点
- **均衡发展**：不要过于集中在一处

#### 2. 进攻技巧
- **活三**：两端都空的三子连线，威胁最大
- **活四**：两端都空的四子，必胜棋型
- **冲四**：一端被堵的四子，需要立即防守

#### 3. 防守技巧
- **及时堵截**：对方形成活三或冲四必须防守
- **以攻代守**：在防守的同时创造自己的威胁
- **预判对手**：提前计算对手可能的进攻路线

### 中级技巧

#### 1. 双三战术
- 同时形成两个活三
- 对手无法同时防守
- 禁手规则下黑方不可用

#### 2. 四三组合
- 同时形成冲四和活三
- 对手必须防守冲四
- 然后活三成为必胜棋型

#### 3. 连环攻势
- 连续制造威胁
- 迫使对手不断防守
- 最终形成无法防守的棋型

### 高级技巧

#### 1. 禁手陷阱（仅限白方）
- 诱导黑方进入禁手位置
- 在禁手规则下，黑方被迫避开
- 利用规则获得优势

#### 2. VCF战术（Victory by Continuous Four）
- 连续不断的冲四
- 强制对手防守
- 最终形成必胜棋型

#### 3. VCT战术（Victory by Continuous Three）
- 连续不断的活三威胁
- 对手被迫防守
- 最终形成四三或双三

### AI对战策略

#### 对抗轻松AI
- 可以大胆进攻
- AI容易忽略威胁
- 适合练习连五技巧

#### 对抗进阶AI
- 需要注意防守
- AI会主动进攻
- 学习中级策略

#### 对抗大师AI
- 必须深思熟虑
- 每一步都很关键
- 需要高级策略和计算

## 常见问题

### Q1: 为什么不能直接双击index.html打开？

**A**: 游戏使用了现代ES Modules技术，必须通过HTTP服务器运行。请使用`npm run serve`或运行`启动游戏.bat`。

### Q2: 如何悔棋？

**A**: 点击游戏下方的"悔棋"按钮。人人对战撤销1步，人机对战撤销2步。

### Q3: 禁手规则是什么？

**A**: 禁手规则下，黑方不能形成三三、四四或长连（6子以上）。白方无限制。这是专业五子棋规则。

### Q4: AI太难了，怎么赢？

**A**:
- 降低AI难度到"轻松"
- 使用提示功能辅助
- 多练习基础技巧
- 学习开局布局策略

### Q5: 如何保存游戏进度？

**A**: 目前游戏不支持存档功能。但可以：
- 使用截图保存棋盘状态
- 查看棋谱记录复盘
- Steam版本将支持云存档

### Q6: 游戏支持哪些语言？

**A**: 目前支持简体中文和英语。未来将支持更多语言。

### Q7: 可以在线对战吗？

**A**: 当前版本仅支持本地对战。在线对战功能计划在v1.1版本推出。

### Q8: 棋盘尺寸怎么选？

**A**:
- **15×15**：标准五子棋棋盘，推荐使用
- **19×19**：围棋棋盘，适合更复杂的对局

### Q9: 什么是"最后一手"标记？

**A**: 棋盘上带黄色光圈的棋子是最近一步落子，帮助玩家快速定位最新棋局变化。

### Q10: 如何查看棋谱？

**A**: 游戏右侧的"棋谱记录"面板实时显示所有落子，格式为"手数. 黑子 白子"。
